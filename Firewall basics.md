# 🔐 Firewall Basics & Concepts

## 📌 Introduction
A **firewall** is a network security system that monitors and controls incoming and outgoing traffic based on predefined security rules. It acts as a barrier between a trusted internal network and untrusted external networks like the internet.

---

## 🎯 Objectives
- Understand the concept of firewalls  
- Learn different types of firewalls  
- Explore how firewalls work  
- Identify their importance in network security  

---

## 🔐 What is a Firewall?
A firewall is designed to:
- Allow legitimate traffic  
- Block unauthorized or harmful access  
- Protect systems from cyber threats  

---

## ⚙️ Types of Firewalls

### 1. Packet Filtering Firewall
- Filters traffic based on:
  - IP address  
  - Port number  
  - Protocol (TCP/UDP)  
- Fast but provides basic security  

---

### 2. Stateful Firewall
- Tracks active connections  
- Makes decisions based on session state  
- More secure than packet filtering  

---

### 3. Proxy Firewall
- Acts as an intermediary between user and internet  
- Hides internal IP addresses  
- Provides additional security layer  

---

### 4. Next-Generation Firewall (NGFW)
- Advanced firewall with features like:
  - Deep packet inspection  
  - Intrusion prevention  
  - Application awareness  

---

## 🔄 How Firewall Works

1. Data (packet) tries to enter or leave network  
2. Firewall checks rules (ACLs – Access Control Lists)  
3. Evaluates:
   - Source IP  
   - Destination IP  
   - Port number  
   - Protocol  
4. Decision:
   - Allow  
   - Deny  

---

## 🧠 Key Terms

- **IP Address:** Unique identity of a device  
- **Port:** Logical communication endpoint (e.g., 80, 443)  
- **Protocol:** Rules for communication (TCP/UDP)  
- **ACL (Access Control List):** Set of firewall rules  

---

## 📊 Common Ports

| Port | Service |
|------|--------|
| 80   | HTTP   |
| 443  | HTTPS  |
| 21   | FTP    |
| 22   | SSH    |
| 25   | SMTP   |

---

## 🏠 Uses of Firewall

- Protect home networks  
- Secure school/office systems  
- Prevent unauthorized access  
- Control network traffic  

---

## ⚠️ Advantages

- Enhances security  
- Blocks malicious traffic  
- Prevents unauthorized access  
- Helps in monitoring network activity  

---

## ❌ Limitations

- Cannot stop all attacks  
- Does not fully protect against internal threats  
- Requires proper configuration  

---

## 🔍 Real-World Example

A school firewall:
- Allows browsing (ports 80 and 443)  
- Blocks unauthorized incoming connections  
- Restricts harmful or non-educational content  

---

## 🏁 Conclusion

A firewall is an essential component of network security. It provides strong protection against external threats but should be combined with other security measures for complete protection.
